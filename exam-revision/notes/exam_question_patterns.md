# Question Pattern and Rubric Map

Use this file to capture repeatable question patterns, scoring criteria, traps, and grading language.

Keep this generic enough to guide altered practice. Do not paste large copyrighted rubrics or full source exams into a public repository unless you have the right to redistribute them.

## Status Legend

- `unopened`: identified but not practiced.
- `open`: in the current practice pool.
- `repair`: attempted with missed rubric points.
- `test_stage`: passed once and needs a later blind test.
- `closed`: passed practice and later blind test.
- `blocked`: missing source material or user decision.
