# Run Files

Practice runs are Markdown answer sheets.

Lifecycle:

1. An agent creates a run in `active/`.
2. The learner writes answers directly under `Answer` headings.
3. The agent grades in place without overwriting answers.
4. The agent updates the relevant ledger in `study-ledgers/`.
5. The agent moves the graded run to `completed/`.

Use `_templates/run-template.md` as the starting shape for new runs.
