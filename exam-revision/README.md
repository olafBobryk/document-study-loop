# Exam Revision Workspace

This folder is the default working area for a Document Study Loop.

- `source/`: local-only raw documents such as PDFs and slide decks.
- `notes/`: source maps, rubric maps, and question-pattern notes.
- `runs/active/`: current answer sheets.
- `runs/completed/`: graded or cancelled answer sheets.
- `runs/_templates/`: reusable run file templates.
- `transcripts/`: optional generated transcripts or summaries when safe to keep.
- `diagrams/`: optional generated diagrams used by runs.

Do not use files under `examples/` as live state. Copy patterns from examples into this workspace only when setting up a new loop.
