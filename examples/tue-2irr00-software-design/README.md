# Example: TU/e 2IRR00 Software Design Loop

This folder is a compact example of a Document Study Loop. It is not live study state.

Source context:

- Course/topic area: TU/e 2IRR00 Software Design.
- Source documents: local-only lecture slides and sample exam materials supplied by the learner.
- Public example policy: no raw PDFs, no full transcripts, and no copied sample exam.

Files:

- `example-topic-source-map.md`: tiny source map showing how topics connect to local-only documents.
- `example-ledger.md`: tiny ledger showing topic state.
- `example-run-answered-and-graded.md`: one answered-and-graded Markdown run.

Use this folder to understand the shape of the workflow, then create real live files under `exam-revision/` and `study-ledgers/`.
